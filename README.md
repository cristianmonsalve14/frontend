# frontend

Frontend de la plataforma Libro Digital.

Aplicación web desarrollada con React, TypeScript y Tailwind CSS. Permite la interacción de usuarios con los microservicios del backend, ofreciendo una experiencia moderna y responsiva para la gestión académica.

## Stack tecnológico
- React
- TypeScript
- Tailwind CSS
- Vite

## Instalación y ejecución
1. Clona este repositorio.
2. Instala las dependencias:
   ```sh
   npm install
   ```
3. Inicia la aplicación en modo desarrollo:
   ```sh
   npm run dev
   ```
